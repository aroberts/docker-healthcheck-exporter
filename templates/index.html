Docker Container Health Exporter

This exporter monitors Docker containers with health checks and exposes their status as Prometheus metrics.

Metrics Endpoint

Prometheus metrics are available at: /metrics

Metric Details

The exporter provides the following metrics:

Metric Description Labels
docker_container_health_status Health status of Docker containers with health checks container_id, container_name, image, project, service
docker_container_health_failure_streak Number of consecutive health check failures for Docker containers container_id, container_name, image, project, service

Status Values

The health status is represented by the following values:

  • Unhealthy 0
  • Healthy 1
  • Starting 2
  • No Health Check 3

Configuration

The exporter can be configured using the following environment variables:

Variable Description Default
POLL_INTERVAL Interval in seconds to poll Docker API 15
